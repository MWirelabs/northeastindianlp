# Contributing to Northeast India NLP Catalog

We welcome contributions from researchers, labs, students, and anyone working on NLP for Northeast Indian languages.

## How to Contribute

You can contribute by:
- Opening a **Pull Request** (preferred), or
- Creating an **Issue** with the resource details

## Entry Format

Use the following format:

```markdown
- [Resource Name](direct-link): Short description. Key metric if available. (Your Name / Organization)
```

## Rules

- Link directly to the model or dataset page (HuggingFace, GitHub, or official repository) -- not just a paper.
- Papers without a released model or dataset are not accepted.
- Entries are added on a best-effort basis; we can't independently verify every submission's claims.
